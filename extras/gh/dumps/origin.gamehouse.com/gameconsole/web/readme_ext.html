RealArcade Beta Preview Instructions

Thanks for your interest in testing the RealArcade Beta Preview!
Before you download the RealArcade Beta Preview, there are a few things you should be aware of. Please read this page thoroughly and make sure you're comfortable with it before downloading the RealArcade Beta Preview.  

What is a "Beta Preview?"
The RealArcade Beta Preview is a special version of RealArcade that is near completion, but still has bugs. RealNetworks is offering the Beta Preview to a select group of people to solicit feedback. The Beta Testing process is an important part of creating new products at RealNetworks and helps to ensure that the final product is of the highest possible quality. 

What do I do when I find a bug or want to give feedback?
When you encounter a bug, please take note of how you encountered it and let us know about it at the RealArcade support options web page. For feedback, please fill out the product feedback form. 

Important note about installing games 
Some games (like Demonstar) will offer to "Run the Game" or "Download DirectX" at the end of the installation process. When offered these options, please select "Exit". You'll be able to launch the game from RealArcade.

Important notes about uninstalling games 
When uninstalling a game using RealArcade, please ensure that the uninstaller has actually finished running before clicking "OK" on the dialog claiming "the game uninstall has finished".  The uninstaller for some games (like Monkey Brains) will cause the dialog to appear prematurely.

If you have problems uninstalling a game from within RealArcade, please use the "Add/Remove Programs" control panel in Windows to uninstall the game instead.

Minimum system requirements to run the RealArcade Preview Test:

  • 266 mhz processor
  • 32 MB RAM
  • Windows 98, ME, or 2000 with Internet Explorer 5
  • 300 MB free disk space for RealArcade and new games
  • 56K modem or faster connection

For best results be sure that your computer is connected to the internet when you install RealArcade; as part of its installation process, RealArcade will download the latest edition of the Game Guide and the latest catalog of downloadable games.
