RealArcade Beta Release Notes
Friday, April 20 2001 (build 467)

Top Gotchas

  1. Installing Games Some of the games you download from RealArcade may offer to play the game at the end of the installation. Do not choose this option.
  2. Uninstalling Games Games built using InstallShield will present a dialog that reads "The game uninstall has finished" before the uninstallation is complete. Please wait for InstallShield to finish before clicking "OK"

Reporting Bugs 

  • E-mail RealArcade bug reports to realarcade-defects@real.com (click here to see a sample report)
  • E-mail game-related bug reports to game-defects@real.com

For bug reports, please include information about your system configuration (OS, RAM, CPU speed, video card, PC manufacturer, connection speed) and steps for reproducing the bug. For UI bugs, screenshots would be helpful. Please click here to see some sample bug reports.

  • E-mail general feedback to realarcade-feedback@real.com

Sample Bug Report

To: realarcade-defects@real.com
Subject: RealArcade window should resize if you lower the screen resolution 
My computer:
Dell Dimension L700
700mhz Celeron 64mb RAM
Windows 98 SE
nVidia TNT2 video card
DSL connection

If you resize RealArcade to fill the screen at high resolution (e.g. 1024x768) and then lower your screen resolution (e.g. 800x600), you can't resize RealArcade to that smaller window size without first going back to the higher resolution

1. Set screen to 1024x768
2. Resize RealArcade to be tall enough to fill screen
3. Display Control Panel: change resolution to 800x600
4. Attempt to resize RealArcade window to fit 800x600

Result: can't grab the lower edge of RealArcade window to resize it
Expect: RealArcade to automatically adjust height to be smaller than screen's vertical resolution
