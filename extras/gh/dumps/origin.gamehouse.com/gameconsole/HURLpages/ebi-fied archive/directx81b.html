Upgrade to Latest Version of DirectX
 
If you have DirectX 9.0 installed and RealOne Arcade claims that you have "DirectX 0.0", please read the special note at the bottom of this page.

DirectX is a Microsoft Windows technology that enables high performance graphics and sound on your computer. Some of the downloadable games available in RealOne Arcade may require a newer version of DirectX than the version that you have installed.

Here are two ways to upgrade to the latest version of DirectX:  

  • Use the Microsoft Windows Update website, which detects and updates out-of-date versions of DirectX, drivers, and other Windows software on your computer.
     
  • Visit the official Microsoft DirectX Downloads website, which offers a variety of DirectX-related downloads.

Installing an upgrade of DirectX usually requires you to restart your computer.

Is RealOne Arcade not detecting DirectX 9.0 correctly? Older versions of RealOne Arcade may warn you that you have "DirectX 0.0" If you are experiencing this behavior, please download and install the latest version of RealOne Arcade from www.realonearcade.com, which will fix this problem. We apologize for the inconvenience!
