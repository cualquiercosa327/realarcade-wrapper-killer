Registration Instructions

Once you have completed your purchase of the handheld game, you will receive an 8-character Registration Code in the receipt. To unlock the handheld game using the Registration Code, follow these steps:

1. Start the game on your handheld.

2. Tap the Menu icon to bring up the menu bar.

3. Tap "Options" in the menu bar and select "Register..."

4. Enter the 8-character Registration Code, then tap the "OK" button.

If the Registration Code is correct, you will see this confirmation:

If you see the below message instead, then you may have entered the Registration Code incorrectly. For example, it is common to mistake the number "5" with the letter "S" or the number "0" with the letter "O".

To re-enter the Registration Code, repeat the above process starting from Step 3.

If you are still unable to get your Registration Code to unlock the game, please refer to the "Support and Uninstall" tab on the game's Game Page in RealOne Arcade, or visit http://www.realarcade.com/helpdesk to contact Customer Service.
