Game Requirements Check

Every game has its own definition of what the minimum  requirements are for an enjoyable game experience.   

To help you decide which games will run on your computer, RealArcade compares each game's requirements with your computer's equipment and will let you know if the game may not be ideal for your computer.

Did your computer not meet a requirements check? Click below for more information on how to resolve it:

  • Operating System
  • DirectX version
  • 3D hardware
  • CPU speed
  • Memory (RAM)
  • Free disk space

Operating System

If your computer does not meet the operating system requirements for a game, you may need to upgrade your PC to a newer version of Windows. 

If your operating system is Windows 2000, you may not be able to play games that are written specifically for the "consumer" versions of Windows, i.e. Windows 95, Windows 98, and Windows ME (also known as Millennium Edition). 

In some cases, games that require a "consumer" version of Windows may still work with Windows 2000, but it is possible that it may not function as well.

DirectX version

DirectX is the multimedia system foundation for Windows operating systems and is required for many of our games. If your game requires an update to DirectX, please follow the installation instructions found on the Microsoft download site.

Here are two ways to upgrade to the latest version of DirectX:  

  • Use the Microsoft Windows Update website, which detects and updates out-of-date versions of DirectX, drivers, and other Windows software on your computer.
     
  • Visit the official Microsoft DirectX Downloads website, which offers a variety of DirectX-related downloads.

3D hardware

A computer can be equipped with specialized hardware devoted to 3D graphics. Some computers have this 3D hardware built-in, while others may need to be upgraded with the addition of a 3D graphics card.

For your convenience, here are links to the websites of several popular manufacturers of 3D graphics cards and built-in 3D hardware:

  • ATI
  • Creative Labs
  • ELSA
  • Hercules
  • Intel
  • Matrox
  • Visiontek

Some games can simulate 3D graphics with software, called "software rendering". However, the graphics produced with software rendering are not drawn as fast and detailed as 3D hardware.

CPU speed

If your computer does not meet the minimum CPU speed requirements for a game, your best bet is to upgrade to a new computer with a faster CPU.

In some cases, a computer that does not meet the minimum CPU speed requirements for a game may still be able to run the game acceptably if it has an exceptionally large amount of memory; for example a 300mHz computer with 128mb of RAM may be able to perform as well as a 400mHz computer with 32mb of RAM.

Memory (RAM)

If your computer does not meet the minimum memory requirements for a game, consider adding more memory for your computer. Consult with your computer's manufacturer for more information.

Free disk space

If your computer does not have enough disk space to install or run the game, consider deleting files or uninstalling games or programs that you do not use to free up more space.
